Atelier Cv jest prosta wtyczką do stworzenia swojej prezentacji dla pracodawcy w formie CV, którą możemy pobrać w formacie PDF.

=== Description ===

Wtyczka pozwala w dość prosty i szybki sposób, na utworzenie curriculum vitae i pobrania go do formatu PDF.

== Installation ==

Video https://www.youtube.com/watch?v=jxkwFaCdAvI
Dokumentacja:
https://atelierweb.pl/create_cv/plugins/atelier-cv/
Repozytorium Bitbucket:
https://bitbucket.org/maniek0borsuk88/ateliercv/src/master/

1. Prześlij folder do katalogu `/wp-content/plugins/` i zainstaluj
2. Aktywuj wtyczkę za pomocą menu „Wtyczki” w WordPress
